# Personal Portfolio

This is my personal portfolio website built using React and Tailwind CSS. It showcases my skills, projects, and academic background through a clean and structured interface.

## Features
- Responsive and modern UI
- Smooth navigation between sections
- Structured layout with dedicated sections:
  - About
  - Skills
  - Projects
  - Contact
- Clean and readable code structure

## Tech Stack
- React (Vite)
- Tailwind CSS
- JavaScript
- HTML & CSS
